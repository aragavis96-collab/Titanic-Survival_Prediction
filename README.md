# Titanic-Survival_Prediction🚢

A Machine Learning Project to predict whether a passenger survived the Titanic disaster using the Random Forest Classifier.

## 📌Project Overview

This project is based on the Kaggle **Titanic - Machine Learning from Disaster** competition. The objective is to predict passenger survival using demographic and travel information.

## 📂Dataset

Source://www.kaggle.com/competition/titanic

Files Used:

   -train.csv

   -test.csv

   -gender_submission.csv

## 🛠 Technology Used

-Python

-Pandas

-Numpy

-Matplotlib

-Scikit-learn

-Tableau

-Kaggle Notebook

-Github

## 📊 Data Preprocessing

-Handled missing values

-Encoded categorical features

-Feature engineering

-Created FamilySize

-Created IsAlone

-Created Title feature

-Feature selection

## 🤖 Machine Learning Model

Random Forest Classifier

Model Parameters:

-n_estimators=200

-random_state=42

**Public Score: 0.71291**

```

```

3.Open the notebook.

```

```

4.Run all cells.
